# McLeod Dev Tools

Development environment for McLeod Health

## Installation

- Clone this repository into a folder EX: /dev/dev-tools
- Create a password file for postgres. Rename `example-postgres-password.txt` to `postgres-password.txt` and change the password.
- Run the script `./start.sh`
- On the host system you must port forward port 22 & port 80 & port 443
